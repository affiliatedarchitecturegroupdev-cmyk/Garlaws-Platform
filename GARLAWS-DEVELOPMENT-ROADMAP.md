## Quick Reference

### Technology Stack
```
Frontend:      Angular 19 + NgRx + Nx + Tailwind
Backend:       NestJS + Supabase (PostgreSQL)
High Perf:     Rust, C++, Mojo
AI/ML:         Python (PyTorch), Julia (MLJ)
Real-Time:     Elixir (Phoenix), gRPC
Mobile:        Capacitor, PWA
Infrastructure: AWS (EKS, RDS, S3), Coolify, Terraform
```

- POPIA (Data Privacy)
- SARS VAT 15%
- B-BBEE Level 2 → Level 1
- NHBRC, CIDB, SABS/SANS
- SACAA (Drone Operations)
